# Demo flow

## 1) Preview path with paywall view
1. Paste a premium publisher URL into **Content URL**.
2. Keep **Use licensed publisher URL** turned OFF.
3. Click **Create usage credit**.
4. Click **Open content**.
5. Confirm the label above the preview says:
   - **Publisher returned preview (human paywall view)**

## 2) Licensed AI path with full body view
1. Turn **Use licensed publisher URL** ON.
2. Confirm the URL now includes both query parameters:
   - `via=fairfetch`
   - `license=<selected license>`
3. Click **Create usage credit**.
4. Click **Open content**.
5. Confirm the label above the preview says:
   - **Publisher returned full body (licensed AI view)**

## 3) Receipt and proof copy
1. Review receipt fields after redeem:
   - Reference ID
   - Amount
   - Domain
   - Path
   - Time
2. Click **Copy receipt details** to copy the proof block.
3. Confirm the button shows **Copied** feedback.
